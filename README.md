# weather platform
An instrument-inspired live weather dashboard built entirely with vanilla HTML, CSS, and JS. Leverages WeatherAPI for deep atmospheric metrics, HTML5 Canvas for real-time procedural weather animations (rain, storm, fog), and GSAP for interactive 3D UI card physics. Fully responsive, dark-mode design focused on raw performance.
